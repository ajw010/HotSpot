# HotSpot
Scripts in both Google Earth Engine and Python (Colab) that will lead to hot spot analysis of paddocks

The Soils_wet_tropics..... files are a shape file from Qld Globe of the soils at 1:50000 for the Tully region. These files are imported into Earth Engine as an asset and then the code file is used to display the soils.
